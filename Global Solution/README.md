# 🚀 Projeto Mobile

## Solução Tecnológica Integrada à Indústria Espacial

Aplicativo mobile desenvolvido em React Native utilizando Expo + TypeScript com foco em soluções tecnológicas inovadoras conectadas ao ecossistema espacial.

---

# 🌎 Sobre o Projeto

O objetivo do projeto é desenvolver uma aplicação mobile moderna capaz de resolver problemas reais utilizando:

- dados climáticos
- geolocalização
- APIs externas
- inteligência artificial
- monitoramento em tempo real
- conceitos da indústria espacial

A proposta deve demonstrar como tecnologias espaciais podem gerar impacto positivo na Terra.

---

# 🎯 Objetivo

Desenvolver um aplicativo mobile capaz de:

- monitorar informações em tempo real
- consumir APIs externas
- exibir dashboards e relatórios
- armazenar dados localmente
- fornecer uma experiência moderna ao usuário
- aplicar conceitos vistos em aula

---

# 🛰️ Relação com a Indústria Espacial

O projeto deve possuir relação com temas ligados à economia espacial, como:

- satélites

- sensoriamento remoto
